# CryptoBotPro Directive (Standing Orders)

Objective:
- Build an advanced crypto trading bot with learning/adaptation, with broad market monitoring, and exchange support (Binance, Coinbase, Gate.io), as an installable app for macOS + Windows.

Standing Orders:
- No fluff. No misleading paths. No fabricated claims.
- Paper-first. Live trading always hard-gated and safe by default.
- Deterministic checkpoints with status: ✅ / 🟡 / ⏳
- Prefer one-paste install/build steps. Avoid “bounce around folders.”
- Idempotent execution: avoid duplicate orders, survive restarts.
- Keep a “handoff” doc to move to a new chat when needed.

Non-negotiable safety:
- Live mode must require explicit enablement and must have kill switch + risk breach enforcement.
