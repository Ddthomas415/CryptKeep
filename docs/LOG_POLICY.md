# Transcript and Log Policy

## Current policy
- This repository is currently intended for local personal use.
- Plaintext logs and session transcripts are acceptable for now.
- Logs and transcripts may contain raw prompts, filenames, and outputs.
- Logs and transcripts should not be shared externally without review.

## Current storage
- session transcripts are stored locally
- event logs are stored locally

## Future policy trigger
If this project expands beyond local personal use, the following must be reviewed before broader use:
- retention limits
- redaction/sanitization
- export/sharing rules
- privacy review

## Operational note
Users should clear or avoid sharing local logs/transcripts if they may contain sensitive content.
